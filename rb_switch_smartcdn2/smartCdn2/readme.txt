<b>Smart jQuery CDN (향상된 jQuery 임포트) 스위치 이용안내</b>
------------------------------------------------------------

업데이트 : 2013.07.13 / jQuery 1.10.2 버전으로 업데이트, 하위 버전 호환성을 위해 jQuery Migrate plugin v1.2.1 추가

업데이트 : 2013.07.01 / jQuery 1.10.1 버전으로 업데이트

이 스위치는 킴스큐 기본 헤더 스위치인 jQuery CDN (코드명: cdn)보다 향상된 jQuery 임포트 기능을 제공합니다.


<b>1) 이 스위치는 헤더 스위치로 위치하는게 좋습니다. (마켓을 통한 자동설치 권장)</b>

jQuery 등의 사전 라이브러리 성격은 반드시 head 영역에 미리 호출되어야 오류가 없습니다.

<b>2) 스위치를 새로 설치하신 후엔 반드시 관리자페이지 시스템 &gt; 스위치의 스위치목록에서 "SAVE" 버튼을 한 번 꼭 눌러주어야 정상 작동합니다.</b>

<b>3) 스위치의 기본경로를 유지하십시오.</b>

./switchs/head/smartCdn2/

<b>킴스큐 기본 스위치는 마이크로소프트 CDN서비스(ajax.aspnetcdn.com)입니다. 일반적으로 가장 많이 사용하는 CDN이 좋은 것이니 이것을 구글CDN으로 변경합니다.</b>

<b>더불어 가장 중요한 기능은 CDN이 즉각 응답하지 않을 때는 로컬파일을 호출합니다.</b>
